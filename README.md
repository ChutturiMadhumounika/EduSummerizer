# EDUSUMMERIZER

EDUSUMMERIZER is a simple AI-based web application that helps students understand study material easily. The user can upload a PDF or text file, and the system will automatically generate a short summary and multiple-choice questions (MCQs). The generated content can also be downloaded as a PDF for easy revision.

## Features
- Upload PDF or text files
- Automatic summary generation
- MCQ generation based on the uploaded content
- Download summary and questions as PDF
- Login and registration system
- Simple dashboard interface

## Technologies Used
- Python
- Flask
- HTML and CSS
- SQLite Database
- NLP models
- FPDF

## How to Run
1. Install required libraries
```
pip install flask flask_sqlalchemy pdfplumber transformers fpdf
```

2. Run the application
```
python app.py
```

3. Open the browser and go to
```
http://127.0.0.1:5000
```
